# Sato_Clinic
Webサイト制作報告書

案件名：佐藤医院 Webサイトリニューアル（Topページ制作）

1. 制作概要

本案件では、佐藤医院の既存Webサイトをリニューアルするため、依頼書および仕様書に基づきトップページ（Top.html）を制作しました。
従来のWebサイトは1990年代に作成され、デザインや構造が古く、スマートフォン対応もされていないことが課題となっていました。

依頼書
そのため、本制作では以下の点を目的としてトップページを設計・実装しました。
患者が必要な情報に素早くアクセスできる構造
スマートフォンでも閲覧しやすいレスポンシブ対応
医療機関として安心感・信頼感を与えるデザイン
お知らせなどの情報更新がしやすい構成
また、トップページの掲載文章については、仕様書の内容を改変せずそのまま使用しています。

仕様書
2. トップページ構成
依頼書で指定された構成をもとに、以下のセクションでページを設計しました。
ファーストビュー（ナビゲーション付き）
診療時間
お知らせ
院長挨拶
診療内容（4項目）
当院について
スタッフ紹介
アクセス

フッター
この構成は、依頼書に記載されたトップページ構成の要件に従っています。

3.制作のアピールポイント
-----------------
🔳視認性と安心感を重視したデザイン

医院のイメージに合わせて、白・グリーン・ネイビーを基調とした清潔感のあるデザインを採用しました。
依頼書では以下のデザイン要望が示されています。
白を基調とした清潔感のあるデザイン
淡いグリーンまたはブルーを差し色に使用
親しみやすい印象のサイト
本サイトではそれらを意識し、以下を実現しました。
ヘッダーとフッターにグリーンを使用し統一感を確保
院内の様子が観れるよう提供された写真を全て、スライドで見ることができます。
医療サイトらしい落ち着いた配色
写真を大きく配置し安心感を演出
スタッフの経歴は、あまり強調せず、ツールチップで確認することができます。
簡易Web予約システムのインターフェイスを実装
